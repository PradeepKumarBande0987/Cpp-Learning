# C++ Learning Projects

Welcome to this C++ practice workspace! This collection is built around learning the fundamentals of C++ by creating small, fun, and useful programs.

## What you can learn here

- Basic C++ syntax and program structure
- Variables, input/output, and control flow (`if`, `else`, `switch`)
- Loops (`for`, `while`, `do-while`)
- Functions and modular code design
- Simple logic and problem solving
- User interaction through console programs

## Projects included

### CGPA Calculator
A small program to calculate a student's CGPA based on grades and credit hours. This project is great for practicing:
- reading input from the user
- performing arithmetic operations
- storing values in variables
- displaying results clearly

### Rock Paper Scissor
A fun console game that simulates the classic Rock-Paper-Scissors match. This project is ideal for learning:
- random number generation
- conditional logic
- game loop design
- handling user choices

## Why build small fun projects

Starting with small projects is the best way to learn C++ because it helps you:
- apply theory in a hands-on way
- build confidence quickly
- understand program flow from start to finish
- keep learning enjoyable with real outcomes
